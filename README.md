# React De cero a experto ( Hooks y MERN )

## 20. Calendar App - Aplicación

Una pequeña aplicación hecha en React con Hooks para buscar y mostrar gifs animados.

> Created with Vite2

```sh
npm init @vitejs/app gif-expert-app --template react
```

### Styles
